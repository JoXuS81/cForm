We like to write code on Mac. It gives to us lot of pleasure. In most cases we use Visual Studio Code for HTML editing. Apple's products generally come with a more premium price tag, especially if you're eyeing-up a Windows machine with similar hardware specifications. Apple's Mac lineup has become steadily more expensive over the years, and I don't think that's likely to see a change any time soon.
